## About the AI
The current AI uses the NegaMax search algorithm, enhanced with:
- Alpha-Beta Pruning
- Null-Move Pruning
- Zobrist Hashing
- Iterative Deepening

The evaluation function is still being tuned, and further optimizations are planned to improve response times at higher search depths.
